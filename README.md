# Tira Online E-Commerce Database Management System

## Project Overview
The Tira Online E-Commerce Database Management System is a relational database project designed to manage the operations of an online beauty and personal care shopping platform. The system stores and manages customer details, product information, brands, categories, orders, payments, inventory, delivery status, and customer reviews in an organized and secure manner.

The project demonstrates how a Database Management System (DBMS) helps an e-commerce platform efficiently manage large amounts of data while ensuring accuracy, consistency, security, and fast data retrieval.

---

## Problem Statement
Managing customer, product, order, payment, and inventory data manually is time-consuming and may result in errors, data redundancy, and poor data management. This project provides a structured database solution to improve efficiency and maintain data integrity.

---
